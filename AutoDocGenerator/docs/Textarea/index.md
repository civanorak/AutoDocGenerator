Disabling wordwrap will enable horizontal scroll When set to true, the value contained in the inputbox cannot be edited by the user. Default is false. Fired after the value of the textarea is changed. Parameter is the previous value before the change. If the user is typing, this event will be fired after the user hits enter or if enabled, after a set amount of time. This function will be called even if the value is not actually changed since the the last call. Fired after the value of in the textarea is edited.
