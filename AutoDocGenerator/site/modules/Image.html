basic_Image

Namespace: Containers

This class is a container for image data. It supports different color modes and access to the underlying data through () operator. This object implements move semantics. Since copy constructor is expensive, it is deleted against accidental use. If a copy of the object is required, use Duplicate function. Assign(Byte *newdata, const Geometry::Size &size, Graphics::ColorMode mode)

Returns: void

Assigns the image to the copy of the given data. Ownership of the given data is not transferred. If the given data is not required elsewhere, consider using Assume function. This variant performs resize and copy at the same time. The given data should have the size of widthheightGraphics::GetBytesPerPixel(mode)*sizeof(T_). This function does not perform any checks for the data size while copying it. If width or height is 0, the newdata is not accessed and this method effectively Destroys the current image. In this case, both width and height should be specified as 0.
